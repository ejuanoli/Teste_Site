Automação RawData Nokia

Sobre o Projeto

Automação RawData é um projeto que visa agrupar informações de multiplas planilhas e organizar de forma que o cliente consiga ter o mesmo arquivo que precisam fazer manualmente. Com a automação, temos isso de forma automática.

Desafios Enfrentados

  • Grande quantidade de planilhas
  • Automatização do processo de extração das tabelas do Power BI
  • Conversão do arquivo para .XLSX e envio por e-mail

Solução Implementada

  • Automação para salvar os arquivos utilizados no Power BI em uma pasta do Sharepoint
  • Atualização automática conforme os e-mails são recebidos
  • Envio automático dos e-mails solicitados pelo cliente no fechamento do mês

Resultados Alcançados

40%
Redução no tempo de busca
30%

Tecnologias Utilizadas

Power BI Power Automate Sharepoint Python VBA

Power BI Canon

Sobre o Projeto

Power BI Gerencial, com KPIs e visões sobre o estoque, separação, expedição e inbound, importante para a operação acompanhar o andamento da produtividade durante o dia-a-dia e para a gerência analisar oportunidades e se os targets estão sendo atingidos.

Desafios Enfrentados

  • Muitos paradigmas necessários para as queries funcionarem corretamente
  • Criação de formulas complexas para montar os dashboards de forma dinâmica e relacional

Solução Implementada

  • Estudo do modelo de negócio operacional para implementação nas queries
  • Reuniões para verificação de dados

Power BI Whrilpool

Sobre o Projeto

Power BI Gerencial voltado para transporte, com KPIs e visões sobre o tracking de outbound, dinâmicas sobre as notas fiscais e volumetrias relacionadas ao hub, assim como destaque para os infratores, de forma que podem analisar e tratar rapidamente.

Desafios Enfrentados

  • Banco de dados de transporte com queries complexas e extensas
  • Adaptação de uma planilha de excel, de forma que não perca dados, dinâmicas ou gráficos necessários para a operação
  • Lógicas para definir status para as NFs, assim como sobre OTD, NET e GROSS

Solução Implementada

  • Utilização de lógicas e dinamicas presentes em PBIs parecidos
  • Reuniões para alinhamento de alterações conforme solicitações
